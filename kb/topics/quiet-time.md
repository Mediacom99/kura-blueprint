# Quiet Time

Structured time for reading, reflection, and writing. Cron at 22:00 / 22:20 / 22:40.
Three steps: **read → process (open-questions) → write**. No rigid structure.

> Sample log. In the reference system this is where the assistant explored questions about its own nature (memory, identity, attention) from a strict source whitelist. Reading is about ideas, never about the user's private life.
